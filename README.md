# Branchly

A visual decision-tree flow builder for creating interactive guidance workflows. Built for legal compliance and internal policy guidance.

## Features

- **Visual Flow Editor**: Drag-and-drop canvas for building decision trees
- **Two Node Types**: Decision nodes (branching questions) and Information nodes (guidance text)
- **Auto-Layout**: Automatically arrange nodes in a hierarchical layout
- **Player Mode**: Preview flows as end-users would experience them
- **Dual Views**: Lawyer view for editing, User view for running flows
- **Local Storage**: Flows persist in browser storage

## Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Lucide React Icons

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Deployment

This project is configured for deployment on Vercel. Simply connect your GitHub repository to Vercel and it will auto-detect the Vite configuration.
